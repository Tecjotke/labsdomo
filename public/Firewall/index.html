EL BOSS

Operación Cali — un incidente combinado de 45 minutos.

Endpoint con heartbeat rojo, DNS tunneling activo, login VPN no autorizado desde Europa del Este. Tienes 45 minutos para aislar, contener, recolectar evidencia. La flag final está codificada en Base32 dentro del payload DNS y es única para tu código de participante — no se puede compartir.

  • Aislar host vía Security Heartbeat o regla manual TOP priority.
  • Bloquear C2 con Custom Web Category.
  • Packet capture + Wireshark + Base32 decode para la flag final.
